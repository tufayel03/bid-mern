# Server Folder

This folder contains backend/infrastructure code copied from the project root.

## Included

- `worker/`
- `shared/`
- `data/`
- `scripts/`
- `tools/`
- `docs/api documentation`
- `.env.example`
- `ecosystem.config.cjs`

## Important

- This was created as a **non-destructive split**.
- Original files still exist in their old locations.
- If you want, a second step can fully migrate runtime/scripts to use only `client/` and `server/`.
